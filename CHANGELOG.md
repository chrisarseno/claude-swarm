## [0.1.0] - 2026-02-22

### Added

- Initial public release
- Multi-instance Claude Code orchestration
- Task queue system with priority scheduling
- REST and WebSocket API
- YAML-based workflow automation
- Real-time monitoring and load balancing
- MCP server integration
- Model registry with 24 model profiles
- Intelligent task routing with performance feedback
- Agent bridge for enterprise agent integration
